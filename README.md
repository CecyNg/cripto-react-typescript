# Cripto React TypeScript

Aplicación que permite consultar el precio actual de las principales criptomonedas en tiempo real.  
Desarrollada con React, TypeScript y la API de criptomonedas para obtener datos actualizados, validación de formularios con Zod y manejo global de estado con Zustand.   

##  Tecnologías utilizadas
- React  
- TypeScript  
- Axios  
- Zustand  
- Zod  
- Vite
